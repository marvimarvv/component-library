# Theming

All themes are implemented as CSS custom properties (design tokens). Switching themes is as simple as
applying a class to the root element — no JS theming runtime, no CSS-in-JS overhead.

## Full Token Reference

Dark mode is opt-in per theme and only swaps the background tokens (`--color-background, --color-background-contrast`) which define the page background and body text — the primary palette stays the same across light and dark.
Dark mode styles on the component level are added using the `dark:` prefix of Tailwind, which looks for the presence of the `.dark` class on a parent element.
All CSS custom properties follow a consistent naming convention so they can be used with Tailwind's
`bg-primary-*`, `text-primary-*`, and `bg-primary-*-contrast` utilities.

```css
/* Primary palette */
--color-primary-100
--color-primary-300
--color-primary-500
--color-primary-700
--color-primary-900

/* Foreground on primary */
--color-primary-100-contrast
--color-primary-300-contrast
--color-primary-500-contrast
--color-primary-700-contrast
--color-primary-900-contrast

/* Muted foreground on primary */
--color-primary-100-contrast-muted
--color-primary-300-contrast-muted
--color-primary-500-contrast-muted
--color-primary-700-contrast-muted
--color-primary-900-contrast-muted

/* Page background & body text */
--color-background
--color-background-contrast

/* Component radii */
--border-radius
```

## Gradient Theme

A playful, dreamy aesthetic built around smooth color gradients, soft shadows, and flowing transitions.
The hue is a mid-range violet (250°) with a saturation of 67%. Border radius is generous (`1rem`),
giving all components softly rounded corners.

## Neon Theme

A flashy, high-contrast style inspired by neon signs and synthwave culture. The hue is a vivid green (110°)
with saturation pushed to 80% on the key levels. Border radius is `0rem` — everything is sharp-edged,
mirroring the raw, angular aesthetic of neon signage.
